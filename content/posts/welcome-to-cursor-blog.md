# Cursor Blog에 오신 것을 환영합니다

안녕하세요! Cursor Blog에 오신 것을 환영합니다. 이 블로그는 마크다운으로 작성한 글을 아름답게 렌더링하는 정적 블로그 사이트입니다.

## 블로그의 특징

이 블로그는 다음과 같은 기능을 제공합니다:

- **마크다운 지원**: 간편하게 글을 작성할 수 있습니다
- **태그 및 카테고리**: 글을 체계적으로 분류할 수 있습니다
- **월별 아카이브**: 시간순으로 포스트를 관리합니다
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 경험을 제공합니다

## 마크다운 문법 예시

### 코드 블록

```javascript
function greet(name) {
  return `안녕하세요, ${name}님!`;
}

console.log(greet("방문자"));
```

### 리스트

- 세 번째 항목

### 강조

**굵은 텍스트**와 *기울임* 텍스트를 사용할 수 있습니다.
